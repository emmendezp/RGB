    Ejercicio 1 # Ejercicio 1 Let rgb1 and rgb2 be two rgb colors. What would rgb1 * rgb2 mean? RGB # Se refiere a un sistema para representar los colores que se utilizan en la pantalla de un ordenador. RGB es una combinación de rojo, verde y azul. Estos colores pueden combinarse en varias proporciones para obtener cualquier color del espectro visible. Cada nivel se mide por el rango de números decimales de 0 a 255 (256 niveles para cada color).
    Ejercicio2
    Ejercicio 2 # Ejercicio 2 Develop a terrain visualization application Mach band # Las bandas de Mach son una ilusión óptica que parte de una imagen con dos bandas, una iluminada y la otra oscura, separadas por una estrecha banda central coloreada con un gradiente de iluminado a oscuro. El ojo humano percibe dos estrechas bandas de diferente luminosidad, que no están presentes en la imagen verdadera, a cada lado del gradiente.
    Ejercicio3
    Ejercicio 3 # Ejercicio 3 Research some dither visual apps. Dithering # Es una técnica usada en computación gráfica para crear la ilusión de profundidad de color en imágenes con una paleta de colores limitada. En una imagen tramada, los colores no disponibles en la paleta se aproximan por una difusión de píxeles de color dentro de la gama de colores disponibles. El ojo humano percibe la difusión como una mezcla de los colores dentro de ésta.
